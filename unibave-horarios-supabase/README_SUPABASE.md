UNIBAVE - Protótipo pronto para Supabase & Vercel
================================================

Resumo rápido:
- Projeto preparado para funcionar com Supabase (banco + autenticação + storage).
- Você precisa criar contas gratuitas no Supabase e no Vercel.
- Depois de criar o projeto no Supabase, cole duas chaves no Vercel e publique.

Passo a passo simples
---------------------

1) Crie projeto no Supabase
   - Acesse: https://app.supabase.com and sign up.
   - Create a new project and wait for it to be ready.
   - In Project -> Settings -> API, copy:
     - Project URL (REACT_APP_SUPABASE_URL)
     - anon public key (REACT_APP_SUPABASE_ANON_KEY)

2) In the Supabase dashboard:
   - Storage -> Create a bucket -> name: public-logos -> set Public
   - SQL Editor -> paste and run the file 'supabase_schema.sql' included in this ZIP.

3) Deploy to Vercel (upload ZIP)
   - Create account at https://vercel.com
   - New Project -> Import -> Upload the ZIP generated by me (this file).
   - In Vercel project settings -> Environment Variables, add:
     - REACT_APP_SUPABASE_URL = <Project URL from Supabase>
     - REACT_APP_SUPABASE_ANON_KEY = <anon key from Supabase>
   - Deploy. After build, open the URL.

4) Usage notes
   - Login via email (magic link). The app uses Supabase Auth.
   - Upload logos in Config (will be stored in 'public-logos' bucket).
   - Import .xlsx files in Import page; preview and press 'Salvar import' to save to table 'imports'.
